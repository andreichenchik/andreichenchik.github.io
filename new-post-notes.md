Заметка: Минимальный сетап для хостинга сайд-проектов с Dokku

Идея поста: Создать гайд по настройке максимально простого и дешевого решения для хостинга небольших бэкендов и сайд-проектов.

Проблема:
- Cloudflare Workers и подобные сервисы накладывают много ограничений и требуют изучения их архитектуры
- Для сайд-проектов "на выброс" инфраструктурные сложности убивают мотивацию
- Хочется просто написать короткий Python скрипт, который отдает JSON, и быстро его задеплоить
- Coolify слишком сложен - требует отдельный сервер для управления, сложный интерфейс

Решение - Dokku:
- Самый дешевый VPS + Dokku
- Простое развертывание контейнеров с мини-приложениями
- Идеально для небольших бэкендов под мобильные приложения
- Минималистичный подход без лишних сложностей

Мотивация: Убрать барьеры для экспериментов с сайд-проектами, сделать деплой максимально простым и дешевым.

Структура гайда:
1. Настройка VPS
2. Установка Dokku
3. Деплой первого приложения
4. Базовые команды для управления
